# Kafka

## 简介

### 概述

+ kafka 是一个高吞吐、低延迟的消息中间件。

### 术语

+ Topic 用来对消息进行分类，每个进入到Kafka的信息都会被放到一个Topic下
+ Broker 用来实现数据存储的主机服务器
+ Partition 每个Topic中的消息会被分为若干个Partition，以提高消息的处理效率
+ Producer 消息的生产者
+ Consumer 消息的消费者
