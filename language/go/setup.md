# 环境搭建

## 安装

**VSCode中Go环境搭建**：

- 安装Go
- 安装Go插件
- 更新Go插件依赖包

## 运行

``` bash
# 编译 + 运行
go run main.go
# 编译
go build main.go
# 运行
./main
```
