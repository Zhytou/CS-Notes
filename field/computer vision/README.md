## 数字图像处理基础

数字图像处理主要研究图像的数字化表示、操作和分析。它是传统计算机视觉研究（不涉及深度学习）的基础，也是很多算法和预处理手段的来源。

**图像基础**：

- 图像类型：灰度图像、彩色图像（RGB、HSV等）、二值图像、索引图像
- 图像捕获：采样与量化

**图像增强**：

- 空间域滤波
- 频率域滤波
- 直方图处理和基本的灰度处理

**彩色图像处理**：

- 颜色模型
- 真彩色/伪彩色

**噪声处理**：

- 噪声模型
- 去噪方法
